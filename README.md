 📱 Lista 3era App (Android)

 Descripción
Aplicación móvil híbrida basada en WebView que permite acceder al sistema web Lista 3era desde dispositivos Android.

 ! 🚀 Funcionalidad
La app carga el sistema web alojado en un servidor, permitiendo:
- Registrar emergencias
- Visualizar datos
- Acceder desde cualquier lugar

!🛠️ Tecnologías
- Android Studio
- Java
- WebView
- Hosting (InfinityFree)
