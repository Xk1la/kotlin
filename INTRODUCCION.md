 📱 Lista 3era App - Android

 📌 Descripción
Lista 3era App es una aplicación móvil híbrida que permite acceder al sistema web de registro de emergencias desde un dispositivo Android.

Funciona mediante WebView, conectándose a la plataforma web en línea.


 🎯 Objetivo
Facilitar el acceso al sistema desde el celular, permitiendo registrar emergencias en terreno de manera rápida y sencilla.


 ⚙️ ¿Cómo funciona?
La aplicación carga la página web del sistema dentro de la app, permitiendo:
- Registrar datos
- Visualizar información
- Usar el sistema como si fuera una app nativa


🌐 URL del sistema
https://lista3era.rf.gd


 🛠️ Tecnologías utilizadas
